# Portfolio Summary

## One-Line Positioning

FinancialAuditAI is a local-first financial document review platform that turns audit-style document evidence into traceable fields, deterministic rule results, human review records, and xlsx reports.

## Project Boundary

The project is a portfolio and learning system. It uses synthetic, simulated, desensitized, or public data. It is not a production audit system and does not provide audit, legal, investment, compliance, or accounting advice.

## Tech Stack

- Backend: FastAPI, SQLAlchemy, Alembic, Pydantic.
- Database: PostgreSQL, pgvector.
- Frontend: React, TypeScript, Vite, Ant Design.
- Reports: xlsx export.
- Storage: local filesystem under ignored `local_storage`.
- Testing: pytest and frontend build verification.

## Completed Capabilities

- Task Center and document upload.
- OCR/page text storage.
- Classification and field extraction.
- Procurement, sales, confirmation, interview, and contract review scenarios.
- Deterministic Rule Engine with evidence.
- Audit Workbench.
- Review Center with before/after and audit logs.
- Report Center xlsx export.
- RAG Knowledge Center with citations and no-answer handling.
- Rule Center for version and parameters.
- Controlled Agent Workflow with state machine and tool whitelist.
- Bad Case Center and Evaluation Center.
- RBAC, login, Admin Center, audit log query, upload hardening, and safety scan.
- Final documentation and demo script.

## What It Does Not Claim

- No real customer deployment.
- No production security certification.
- No real audit opinion.
- No legal opinion.
- No investment recommendation.
- No automated contract approval.
- No external confirmation sending or bank integration.
- No speech recognition or automatic factual adjudication.

## Portfolio Description

This project demonstrates full-stack product engineering for an evidence-driven financial document review workflow. The implementation emphasizes traceability, deterministic rules, human review, realistic boundaries, and reproducible local delivery rather than speculative AI autonomy.
