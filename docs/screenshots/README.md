# Screenshot Checklist

No screenshot files are required for Phase 20, and no fake screenshots should be generated. If screenshots are added later, use only synthetic or public demo data.

Recommended screenshots:

- Login page.
- Task Center with a synthetic task selected.
- Document upload and document status list.
- Audit Workbench three-column evidence view.
- Review Center queue and before/after correction view.
- Report Center report history and xlsx generation status.
- Knowledge Center RAG query with citations or no-answer result.
- Rule Center rule detail and dry-run area.
- Agent State Timeline with steps.
- Bad Case Center list.
- Evaluation Center result detail.
- Admin Center users/roles/audit logs.

Do not commit screenshots containing real customer names, real contracts, invoices, confirmations, bank data, interview notes, audit workpapers, API keys, tokens, or passwords.
