# Confirmation Demo Samples

This folder contains hand-written mock confirmation data for Phase 16 demos and tests.

- The data is synthetic.
- It does not contain real counterparties, addresses, bank records, confirmations, seals, signatures, or workpapers.
- It is intended only as a small scenario reference for local demos and tests.
- Real uploads, generated reports, vector indexes, and confirmation evidence files must stay under ignored local paths.
